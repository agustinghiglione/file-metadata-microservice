# File Metadata Microservice

This is my solution for the FreeCodeCamp "File Metadata Microservice" project from the Back End Development and APIs Certification.

## Features

- Upload files via a web form or API
- Extract file metadata: name, type, and size
- Drag-and-drop file upload support
- File size validation (10MB maximum)
- Clean, responsive web interface
