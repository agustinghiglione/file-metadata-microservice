FreeCodeCamp Back End Development and APIs Project

What is this?

This microservice extracts metadata from uploaded files. Upload any file and get its name, type, and size in bytes.

Important Requirements

For FreeCodeCamp tests to pass:

  • The file input MUST have name="upfile"
  • The form MUST use multipart/form-data encoding
  • Response will be JSON with name, type, size properties
